# Find Maximum Area Rectangle in Matrix

This program outputs the rectangle with the biggest area in a matrix given a list of rectangles of different length and width.

## Example 1:

### Input: 
    6 16
    ################
    #..#...........#
    #......#.......#
    #......#.......#
    #......#....####
    #########..#####
    4
    3 3
    4 6
    15 2
    2 7

### Output
    2 7
